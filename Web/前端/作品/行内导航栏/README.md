###### <font color=red>元素显示模型之间的转化</font >

| 属性                    | 效果               |
| ----------------------- | ------------------ |
| `display:block`         | 转化为块级元素，可以设置行高和宽度，换行显示   |
| `display：inline-block` | 转化为行内块级元素，可以设置行高和宽度，并且一行显示 |
| `display：inline`       | 转化为行内元素，一行显示，行高和宽度设置无效 |
