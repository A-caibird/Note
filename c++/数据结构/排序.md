# [排序算法](https://www.cnblogs.com/dll-ft/p/5861210.html)

## 概念

- [ ] 稳定排序

  > 排序过程中，数字之间的交换是否只是存在于相邻的两个数之间？
  >
  > 如果是 -----> 稳定排序，否则 -----> 不稳定排序

- [ ] 最坏情况

  > 待排序序列具有何种特性的时候导致排序算法执行的步骤最多，耗时最大

- [ ] 最好情况

  > 与上面相反

- [ ] 额外空间

  > 排序过程种是否需要开辟额外的空间用于储存和交换数据
